服务器
===
把项目部署到服务器端，折腾历程

### 1、安装node
使用nvm安装node的任意版本
引用自 https://cnodejs.org/topic/598954d9206061d87545c5d3

 *   安装nvm

```
curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.33.2/install.sh | bash
```
nvm安装成功后，关闭当前终端，重新连接 
* 验证安装是否成功
```
nvm --version
```
看到输出版本信息0.33.2表示安装成功

*   查看Node.js版本并安装
```
nvm ls-remote 
nvm install v8.2.1
```
* 测试node 是否安装成功
```
node -v
```
